# Factorio WFC

Редактор сетки фабрики Factorio с перспективой автогенерации через Wave Function Collapse.
Сейчас — каркас: C++ логика, pybind11-мост, минимальный GUI на Dear PyGui.

## Запуск

```powershell
pip install -r requirements.txt
make build
make run
```

## Тесты

```powershell
make test
```
